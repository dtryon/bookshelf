### Install/Run

##### Node via n (requires node > v8.x)

```sh
npm install -g n && n lts
```

##### Install dependencies

```sh
npm install
```

##### Run Tests

```sh
npm test
```

##### Run Linter

```sh
npm run lint
```

##### Run (Development mode)

```sh
npm start
```

Once the server has started, navigate to http://localhost:8181.
